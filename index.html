Tipos de etiqueta en Html y CSS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

Estructura de etiquetas
Etiquetas que solo se aperturan Etiquetas que tienen una apertura y sierre
Este tipo de etiqueta común mente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas común mente se utilizan para encerrar un contenido tales como texto img entre otros.

Lista de etiquetas dirigidas al texto

  • p> /p> se utiliza para colocar un texto
  • h1> /h1> se utiliza para colocar un titulo

Listado de etiquetas dirigidas a objetos

  • img> se coloca imagen
  • header> se coloca titulo

Listado de propiedades css dirigidas a texto:

  • color: #ffffff darle color a cualquier objeto de html
  • font-size 1px le da el tamaño a los textos agregados en css.
  • font-family: asigna tipografía a objetos.
  • text-align: especifíca el lugar dle texto.
  • background-color: le da un color de fondo a objetos.

Etiquetas más utilizadas hasta ahora:

  • Body : Indica la parte del cuerpo del contenido de un documento HTML.
  • Head: La parte superior del documento HTML, es donde podremos indicar los metadatos.
  • div: Un elemento que es usado mayoritariamente para agrupar otros elementos.
  • a: Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href.
  • html: Representa la raíz de un documento HTML o XHTML.
  • !DOCTYPE html: Define que el documento esta bajo el estandar de HTML 5
  • link: Usada para enlazar JavaScript y CSS externos con el documento HTML actual.
  • section: Define una sección en un documento.
  • footer: Define el pie de una página o sección. Usualmente contiene un mensaje de derechos de autoría, algunos enlaces a información legal o direcciones para dar información de retroalimentación.
  • Height: le da una altura a objetos.
  • Width: dexribe el ancho de un objeto.
